# Python Exercises

Collection of Python exercises focused on loops, control flow, and user input handling.

## Exercises

### Loop Through Word
Practice iterating through a string using both `for` and `while` loops.

[View Exercise](https://github.com/orelbutbul13/Python-Learning-Projects/blob/main/Exercises/Loop_Through_Word.py)

### Even Numbers Generator
Prints even numbers from 100 to 140 using both `while` loops and `for` loops with `range()`.

[View Exercise](https://github.com/orelbutbul13/Python-Learning-Projects/blob/main/Exercises/Even_Numbers.py)

### Passphrase Checker
Demonstrates user input validation and loop control by repeatedly prompting for a passphrase until the correct value is entered.

[View Exercise](https://github.com/orelbutbul13/Python-Learning-Projects/blob/main/Exercises/Passphrase_Checker.py)

## Skills Demonstrated

- For Loops
- While Loops
- Conditional Logic
- User Input Handling
- String Manipulation
- Problem Solving
