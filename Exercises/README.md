# Python Exercises

Collection of Python exercises focused on programming fundamentals, loops, control flow, list manipulation, and user interaction.

## Exercises

### Loop Through Word
Practice iterating through a string using both `for` and `while` loops.

[View Exercise](./Loop_Through_Word.py)

### Even Numbers Generator
Prints even numbers from 100 to 140 using both `while` loops and `for` loops with `range()`.

[View Exercise](./Even_Numbers.py)

### Passphrase Checker
Demonstrates user input validation and loop control by repeatedly prompting for a passphrase until the correct value is entered.

[View Exercise](./Passphrase_Checker.py)

### Driver List Management
Simulates a racing leaderboard while demonstrating common Python list operations including indexing, updating values, adding and removing elements, list slicing, and generating a final leaderboard.

[View Exercise](./Driver-List-Management.py)

## Skills Demonstrated

- For Loops
- While Loops
- Conditional Logic
- User Input Handling
- String Manipulation
- List Operations
- List Indexing
- List Slicing
- Data Structures
- Problem Solving
