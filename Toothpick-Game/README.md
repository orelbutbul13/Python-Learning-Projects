# Toothpick Game

Command-line strategy game built using Python.

## Overview

This project is a two-player command-line game where players take turns removing 1 to 3 toothpicks from a pile of 21 toothpicks.

The game continues until all toothpicks have been removed. The player who takes the last toothpick wins.
