# Toothpick Game

Command-line strategy game built using Python.

## Overview

This project allows two players to take turns removing 1 to 3 toothpicks from a pile of 21 toothpicks.

The player who removes the last toothpick wins the game.

## Features

- Two-player gameplay
- User input handling
- Input validation
- Game loop logic
- Win condition detection
- Visual toothpick display

## Skills Demonstrated

- Variables and Data Types
- User Input Handling
- While Loops
- Conditional Statements
- Input Validation
- Game Logic
- Program Flow Control
