# Todo List CLI

Command-line Todo List application built using Python.

## Overview

This project allows users to manage daily tasks directly from the terminal. Users can add new tasks, mark tasks as completed, and track their progress throughout the day. When the program exits, a summary of completed tasks is displayed.

## Project Files

- [View Source Code](./Todo_List.py)

## Features

- Add new todo items
- View active tasks
- Mark tasks as completed
- Track completed tasks
- Display a daily completion summary
- Input validation for task selection
- Command-line interface
