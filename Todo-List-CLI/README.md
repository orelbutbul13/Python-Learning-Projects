# Todo List CLI

Command-line Todo List application built using Python.

## Overview

This project allows users to manage daily tasks directly from the terminal. Users can add new tasks, mark tasks as completed, and view a summary of completed tasks before exiting the program.

## Project Files

- [View Source Code](./todo_list.py)

## Features

- Add new todo items
- View active tasks
- Mark tasks as completed
- Track completed tasks
- Display a daily completion summary
- Command-line interface
